# 🧠 AI Learning Productivity Platform

A browser-based study companion that turns raw notes into structured, exam-ready material — built entirely with vanilla HTML, CSS, and JavaScript, with zero backend or database dependency.

## ✨ Overview

Studying often means juggling scattered notes, no clear summaries, and no way to test yourself on what you've actually learned. This platform solves that by giving students one place to capture, organize, summarize, and quiz themselves on their own study material — all running client-side in the browser.

## 🚀 Features

- **Voice-to-Text Input** — Dictate notes hands-free using the Web Speech API
- **Auto-Summarization** — Generates concise summaries from longer study content
- **Key Point Extraction** — Pulls out the most important ideas from any text
- **Quiz Generator** — Creates self-assessment questions directly from your own notes
- **Mind Map Builder** — Visualizes topics and their connections
- **Text Analytics** — Instantly shows word count, estimated reading time, and content difficulty level
- **Study Timer & Streaks** — Tracks focused study sessions and daily consistency
- **Achievements & Ranks** — Gamified badges to keep motivation high
- **Task Management** — Simple to-do system for study goals
- **Analytics Dashboard** — Visual overview of study habits over time
- **Export Options** — Download generated notes as TXT or PDF
- **Persistent Storage** — All data saved locally via Browser Local Storage — no login, no server, no data leaves your device

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Structure | HTML5 |
| Styling | CSS3 |
| Logic | JavaScript (Vanilla) |
| Voice Input | Web Speech API |
| Data Storage | Browser Local Storage |

## 📸 How It Works

1. Enter or dictate your study notes
2. The platform analyzes the text and generates a summary, key points, and a quiz
3. Track your study time, streaks, and progress on the dashboard
4. Export your notes anytime as TXT or PDF

## ▶️ Getting Started

No installation or setup required.

```bash
git clone https://github.com/priyankasuresh05/AI-Notes-Generation.git
cd AI-Notes-Generation
```

Then simply open `index.html` in any modern browser.

## 🎯 Why This Project

Built solo to explore how far a purely front-end, no-backend architecture can go in solving a real productivity problem — while practicing core JavaScript concepts like DOM manipulation, browser APIs, and client-side data persistence.
